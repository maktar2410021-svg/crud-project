# CRUD User Management System

This is a Flask based CRUD web application using mySQL for managing users.

## Features
- Add User
- View User List
- Edit User
- Delete User
- Email duplication prevention using UNIQUE constraint

## How to Run

1. Install Python
2. Install requirements
   pip install flask mysql-connector-python
3. Import crud_app_db.sql into MySQL
4. Run
   python app.py
5. Open browser:
   http://127.0.0.1:10000/users
